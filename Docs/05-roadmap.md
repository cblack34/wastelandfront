# 05 · Roadmap

_Direction, not a schedule. `RESEARCH.md` holds the actual prioritized backlog; this doc explains why those priorities exist. Phases overlap._

## Phase 1 — Core reference (the trust foundation)

Every high-traffic "what should I do" question answered with verified specifics: stats & combat mechanics, per-store buy guides, the upgrade investment roadmap (gear/EW/Wall of Honor/drone), hero & squad meta, and the VS + Arms Race weekly loop. This phase is the site's credibility; everything else builds on it.

## Phase 2 — Calculators

Interactive tools with independently validated data (see [`04-calculators.md`](04-calculators.md)). Start with inventory-shaped tools (chests, speedups), then the researched gap list: troop training, power growth, VS planning. Calculators are the strongest repeat-visit and backlink drivers in this niche.

## Phase 3 — Event & season depth

Per-event deep pages (Desert Storm, Zombie Siege/Invasion, Capitol Conquest, SvS) and **season week-by-week checklists** (S1 The Crimson Plague → S6 Lost Rainforest) — the highest-demand recurring content format in the niche. Includes the server-timeline / day-by-day unlock calendar.

## Phase 4 — Freshness & reach

Evergreen SEO surfaces that need a maintenance loop: gift/redeem codes page, awakening schedules, patch-driven re-verification passes (update "Last verified" stamps), and the topics nobody covers well — spotted gaps include lethality/severe-wound mechanics documentation and newly added stores. Domain cutover to wastelandfront.com.

## Ongoing engineering

Site-wide a11y pass (tables, QuickSummary heading level), Lighthouse budgets, and whatever the review loop keeps surfacing — tracked in `RESEARCH.md` → Inbox.
